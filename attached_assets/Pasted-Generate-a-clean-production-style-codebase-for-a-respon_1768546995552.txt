Generate a clean, production-style codebase for a responsive web app called “Zoom Planner”.

Primary objective:
Create a well-structured React + TypeScript project that I can run locally, modify easily, and deploy anywhere (not tied to Replit hosting). Prioritise maintainable code architecture over “one-click demo” behaviour.

Stack:
- React + TypeScript using Vite
- React Router
- Firebase Web SDK:
  - Firebase Authentication (email/password)
  - Cloud Firestore
- Styling: Tailwind CSS (or simple CSS modules if easier, but keep the UI clean and minimal)
- Client-only app (no custom backend server)

Architecture requirements:
- Firebase config and initialization must live in a single module:
  - /src/lib/firebase.ts
- Create a small data/service layer that is the only place that talks to Firestore:
  - /src/services/tasks.ts
  - /src/services/goals.ts
  - /src/services/habits.ts
- Shared types in:
  - /src/types/*
- Reusable UI components in:
  - /src/components/*
- Route-level pages in:
  - /src/pages/*
- Use React hooks + local state where possible. Avoid heavy global state frameworks unless necessary.

Firestore data structure (store everything per-user):

users/{uid}/goals/{goalId}
- title (string)
- description (string, optional)
- timeframe ("year" | "month" | "week")
- startDate (YYYY-MM-DD string, optional)
- endDate (YYYY-MM-DD string, optional)
- status ("active" | "paused" | "done")
- parentGoalId (string, optional)   // allows Year -> Month -> Week linking
- createdAt (timestamp)

users/{uid}/tasks/{taskId}
- title (string)
- date (YYYY-MM-DD string)
- status ("todo" | "doing" | "done")
- linkedGoalId (string, optional)
- notes (string, optional)
- isPriority (boolean, default false)   // used for weekly priorities
- createdAt (timestamp)

users/{uid}/habits/{habitId}
- name (string)
- isActive (boolean)
- schedule ("daily" | "weekdays")
- createdAt (timestamp)

users/{uid}/habitLogs/{logId}
- habitId (string)
- date (YYYY-MM-DD string)
- completed (boolean)

Navigation structure:
Tabs or sidebar navigation:
- Day

- Year
- Goals
- Habits
